# Switch

Just simple switch function for python

## Usage

```python
from switch import *

# value = switch(variable,array)
# variable = variable you want to switch
# array = [[value,function_name]["default",function_name]

value = switch(x,array)
print(value)

function = switch(x,array)
function()
```
